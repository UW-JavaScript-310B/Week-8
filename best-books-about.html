What is this site?

A web development project focused on the use of external APIs. This project is part of the curriculum for CS310 at the University of Washington.

The goal of this project was to get comforable working with APIs. The base requiremets were to connect to the NYT books API and all users to submit a year, month, and date for which we would retrieve the best seller list.

For my submission, I chose to build out the application into a Bootstrap based website.

What APIs were used for this site?

The following APIs were used to produce the content shown on this site. Thanks to those involved in their development.

New York Times API

Provided lists of best selling books for a specified date.

Open Library API

Utilizing the ISBN number provided by the NYT API, we can retrieve the book cover from the Open Library API.

Random User Generator API

To add some user reviews, we are retrieving head shots and names from the Random User Generator API.

Lorem Bacon API

Rather than writing placeholder reviews or hard coding lorem ipsum, it's more fun to tap into the Lorem Bacon API for our text.
